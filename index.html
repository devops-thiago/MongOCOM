Welcome to MongOCOM Project Page.

Mongo Object-Collection Mapper (MongOCOM) is an Object-oriented document mapper for mongoDB.

Make your code readable.

Using simple annotation concepts you can model your database easily, like this:

package com.example.collections;

import com.arquivolivre.mongocom.annotations.Document;
import com.arquivolivre.mongocom.annotations.ObjectId;

@Document
public class Contact {

    @ObjectId
    private String id;
    private String name;
    private String email;

    //getters and setters go here.
}

Easy to use.

MongOCOM is easy to use, no XML or complex configuration needed, just set up your project, add the library to the class path and start coding. 

package com.example.main;

import com.example.collections.Contact;
import com.arquivolivre.mongocom.management.CollectionManager;
import com.arquivolivre.mongocom.management.CollectionManagerFactory;

public class Example {

    public static void main(String[] args){
        CollectionManager cm = CollectionManagerFactory.setup();
        Contact myFirstContact = new Contact();
        myFirstContact.setName("My Name");
        myFirstContact.setEmail("my_email@server.com");
        cm.insert(myFirstContact);
        System.out.println("Contact _id: " + myFirstContact.getId()
                                           + " inserted successfully.");
    }

}
